######################
Matrices et DataFrames
######################

.. index:: dataframe, matrice

Le langage python sert entre autres à faire des calculs numériques.
Deux extensions sont maintenant incontournables tant elles
simplifient l'écriture de calculs matricielles et la
manipulation des données. La librairie :epkg:`numpy`
permet de manipuler les matrices, :epkg:`pandas`
pour la manipulations de données avec les *DataFrames*.
